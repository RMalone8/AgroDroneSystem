# AgroDrone Frontend

#### Before running the frontend, you must create a .env file with the following:
* VITE_DEVICE_TOKEN="uuid-for-authorization-with-backend"
* VITE_BACKEND_URL="http://backend_url:port"

The backend corresponds with [Cloud-to-Pi-Communication](https://github.com/RMalone8/Cloud-to-Pi-Communication)

Once the backend is active, run project for local dev with:

`npm run dev`
